# 世界の図書館 / World Library

## 概要
「世界の図書館」は、  
国家・大学・組織に閉じないかたちで、  
**知・本・対話・場所をゆるやかにつなぐ構想**です。

これは完成したプロジェクトではありません。  
問いを持ち、試行錯誤し、記録し続けるための「図書館的実践」です。

---

## なぜ「世界の図書館」なのか
現代において、知は次のような場所に分断されています。

- 専門知は専門家の内側に
- 実践知は現場の外に見えにくく
- 個人の思索は一過性の発信で消えていく

世界の図書館は、それらを  
**保存し、翻訳し、再び現場へ返す回路**をつくろうとしています。

---

## 基本的な考え方
- 図書館は「建物」ではなく「機能」
- 知は完成形よりも「判断のための素材」
- 中央集権ではなく、複数の小さな拠点
- 専門と市民、理論と実践の往復
- 未完成であることを前提に公開する

---

## このGitHubの役割
このリポジトリは、世界の図書館構想における  
**定義・記録・更新のための保存庫**です。

ここに置かれるもの：
- 構想の定義文
- 運営原則・編集方針
- 試行錯誤のログ
- 公開可能な草稿やメモ

ここに置かれないもの：
- 個人情報
- 非公開の対話内容
- 商用契約に関わる内部資料

---

## noteとの関係
- note：思考の途中、物語、現場の記録
- GitHub：定義、整理、版管理された文章

同じ内容が、異なる言葉で書かれることがあります。  
それは矛盾ではなく「翻訳」です。

note（外向き）  
→ GitHub（整理・固定）  
→ 現場（再実装）

この循環を大切にしています。

※ これは一例であり、モデルの固定を目的としません。

## 関わり方について
この構想は、参加者を「募集」するものではありません。

- 読む
- 批判する
- 翻訳する
- 別の土地で試す
- 記録を残す

いずれも、立派な関わり方です。

何かを始める前に、  
まず「読んでから考える」ことを歓迎します。
